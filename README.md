# 台股籌碼動能雷達

Streamlit 產業籌碼分析工具，可依集保原始持股級距自訂四類投資人門檻。

## 本版功能

- 網站密碼保護，預設密碼：`zhuyi`
- 自訂散戶／中實戶／大戶上限，預設為 50／400／1000 張
- 門檻必須依序遞增，錯誤時會停止分析並顯示提示
- 可切換主要分析群組：散戶、中實戶、大戶、超級大戶
- 泡泡圖、產業排行榜、個股篩選與四類持股走勢會依設定同步重算
- 比較期間支援最近 1／4／13／52 個資料週平均

> 集保資料以固定持股級距發布，因此介面只提供原始級距邊界，避免拆分級距造成不準確的估算。

## 執行方式

```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

開啟網站後輸入密碼 `zhuyi`。

## 資料目錄

├─ tdcc/  # 集保股權分散表 CSV

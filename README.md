# 台股籌碼輪動雷達

Streamlit 研究平台，整合集保股權分散表、TEJ 歷史資料與 XQ／Total Pool。

## 最新版重點

- 月營收年增率可從 XQ `月營收年增率` 欄位直接讀取。
- 比較基準改為期間平均：
  - 前 1 週：最近 1 個歷史資料週。
  - 前 1 月：最近 4 個歷史資料週平均。
  - 前 1 季：最近 13 個歷史資料週平均。
  - 前 1 年：最近 52 個歷史資料週平均。
- 持股結構分成四層並提供個股歷史走勢：
  - 散戶：50 張以下。
  - 中實戶：50–400 張。
  - 大戶：400–1000 張。
  - 超級大戶：1000 張以上。
- 資料檔案名稱、大小或修改時間變化後會自動刷新快取。

## 安裝與啟動

```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## 每週更新資料

```text
data/
├─ tdcc/  # 新增最新集保 CSV，保留歷史檔
├─ tej/   # TEJ 歷史 xlsx
└─ xq/    # 最新 Total Pool CSV
```

Total Pool 建議至少包含：`代碼`、`商品`、`成交`、`總量`、`月營收年增率`、`細產業`。

TEJ 與集保日期重疊時優先採用集保。缺少歷史基期的股票不補 0，也不納入變化中位數。

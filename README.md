# 台股籌碼動能雷達

Streamlit 產業籌碼分析工具，可依集保原始持股級距調整散戶上限與大戶下限。

## 可調整標準

- 散戶：持股張數小於或等於「散戶上限」，預設 50 張以下。
- 大戶：持股張數大於或等於「大戶下限」，預設 400 張以上，包含 1000 張以上。

因此使用預設值時，「大戶」包含 400～1000 張以及 1000 張以上的全部持股。

門檻限用集保原始級距邊界，避免拆分級距產生不準確的估算。泡泡圖、排行榜、個股篩選與持股走勢會依門檻同步重算。

為避免每次調整選項都重新運算，請完成條件設定後按「套用分析設定」；相同條件的分群結果會自動快取。 快取最多保留兩組小型摘要，個股走勢只重算目前選取的股票，避免 Streamlit Cloud 記憶體持續增加。

## 執行方式

```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

本版不需要登入密碼。
